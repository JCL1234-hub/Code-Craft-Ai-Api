# Universal AI API v1.0.0

A Production-Ready AI API fully built using Node.js, Vercel Serverless Functions, and the **Google AI Studio (Gemini)** suite. 

**Developer:** @username_506

## 🚀 Features

- **No Express Server** — Pure Vercel Serverless Functions.
- **Auto-updating Stats** — Tracks API usage inside `stats.json`.
- **CORS Enabled & Secured** — Built-in basic Rate Limiting and Maximum Prompt length validation.
- **Multimodal** — Supports Chat, Code Generation, Imagen 3 Image Gen, and Gemini 2.0 Flash TTS Native Audio.
